Ruby

  • Creator: Yukihiro “Matz” Matsumoto
  • Date Created: Publicly released in 1995

  • Matsumoto created Ruby as a blend of his favorite languages: Perl, Smalltalk, Eiffel, Ada, and LISP. Ruby is a blend of functional and imperative programming. Here are some of the main characteristics of Ruby: everything is an object, it is a flexible language, it has exception handling features, it includes a garbage collector, and enables OS independent threading. Ruby is mainly used for web development, DevOps, web scraping, and data processing. It is still widely used and remains in the top ten list of most popular programming languages according to indices such as the TIOBE index.
