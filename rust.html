Rust

  • Creator: Graydon Hoare/Mozilla
  • Date Created: First stable release - 2015

  • In 2006, Graydon Hoare was the initial developer of Rust while working on it as a personal side project at Mozilla Research. In 2009 it was officially sponsored by Mozilla. Many large companies such as Amazon, Discord, Dropbox, and google use Rust. Rust is used by 2.8 million coders worldwide according to MIT Technology Review. Rust enables coders to write small, fast code without memory bugs which significantly reduces the amount of software crashes. Rust can be used for anything from building command line tools, WebAssembly, networking services, and embedded systems software. Some recent use cases of Rust include Dropbox using it to sync files to your computer, and Cloudflare using it to “process more than 20% of all internet traffic” according to MIT Technology Review.
