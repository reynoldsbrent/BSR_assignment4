C# (C Sharp)

  • Creator: Microsoft
  • Date Created: Released in 2002 alongside Visual Studio IDE

  • C# is now 22 years old with the latest version being C# version 11. It is an open-source, cross-platform, and object-oriented programming language that runs on the .NET Framework and consistently places within the top 5 programming languages on GitHub. According to Microsoft, over 5 million developers use C# and it is used at many large companies such as GE Aviation, Chipotle, UPS, Stack Overflow, and Alaska Airlines. C# can be used for creating Desktop, Web, Cloud, Mobile, Gaming, IoT, and AI applications. It is typically used to develop the backend infrastructure of these applications.
